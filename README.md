# ☕ POS Cafe Analytics  
**Capstone Project – University of Southern Queensland (Feb – May 2025)**  

---

## Project Overview  

POS Cafe Analytics is an end-to-end data analytics project designed to simulate a Shopify-based café environment and transform transactional data into actionable business insights.  

The objective of this project was to:
- Analyse sales patterns and customer behaviour  
- Forecast daily and seasonal sales  
- Visualise ingredient consumption trends  
- Design an intuitive, decision-ready analytics dashboard  

This project demonstrates the integration of data engineering, exploratory analysis, machine learning, and data visualisation into a real-world business solution.

---

## Problem Statement  

Cafés operating on Shopify POS systems generate large volumes of transactional data but often lack analytical infrastructure to:

- Identify high-performing products  
- Forecast demand based on seasonality  
- Analyse weather-driven trends  
- Monitor ingredient consumption efficiently  

This project addresses these challenges through structured analytics and predictive modelling.

---

## Tech Stack  

- Python (pandas, NumPy, scikit-learn)  
- Power BI  
- Linear Regression (Sales Forecasting)  
- Streamlit (Proposed Deployment)  
- Simulated Shopify API data (CSV-based datasets)  

---

## Data Processing & Preparation  

- Cleaned and transformed café transaction datasets using pandas  
- Handled missing values and inconsistent formatting  
- Engineered features including:
  - Day-of-week  
  - Temperature  
  - Holiday indicators  
  - Seasonal categories  
- Structured datasets for both machine learning modelling and dashboard integration  

---

## 🤖 Predictive Modelling  

A Linear Regression model was implemented to forecast daily sales using:

- Temperature  
- Day-of-week  
- Holiday flags  
- Seasonal indicators  

The model captured seasonal patterns and provided interpretable outputs for business decision-making.

## Dashboard Design (Power BI)
